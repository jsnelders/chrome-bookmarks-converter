Chrome Bookmarks Converter

v1.0.0 (14 Novermber 2019)

Raw and rough and hacked together over the last couple of hours to get the basics working. More refinement to come.

Place a file named "bookmarks.html" in the same directory and select "Convert".
For large HTML files, give the browser a few seconds to display after the complete alert shows.
NOTE: You'll need to run "run.html" under http://localhost, not directly from the file system (e.g. file:///run.html) - Chrome will spit the dummy.
